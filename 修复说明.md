## 常见问题

### Q: 如何使用绘图工具？
A: 
1. 鼠标悬停在图表左侧，会显示绘图工具栏
2. 点击"水平线"图标，然后在图表上点击即可绘制水平线
3. 点击"趋势线"图标，在图表上点击两次绘制趋势线
4. 点击"垃圾桶"图标清除所有绘图

### Q: 如何设置止盈止损？
A:
1. 选择一只股票
2. 在右下角"模拟(Sim)"面板输入成本、股数、止盈、止损
3. 图表上会自动显示对应的价格线
4. 当价格触及止盈/止损时，面板会闪烁提示

### Q: 如何管理持仓？
A:
1. 在"模拟(Sim)"面板设置好交易参数
2. 点击"加入持仓组合"按钮
3. 切换到"持仓(Assets)"标签查看所有持仓
4. 持仓会自动更新价格和盈亏

### Q: 如何切换策略？
A: 在顶部工具栏选择不同的策略（Holo、CRT、Envelope、T3），图表指标和SOP指令会自动切换。

### Q: 如何停止服务器？
A: 在终端按 Ctrl+C，或关闭终端窗口。

### 问题4: 主应用显示"发生未知错误"
**错误**: 页面显示"发生未知错误，请刷新页面"

**原因**: index.html的`<body>`标签使用了Vue指令`:data-theme`，但Vue还未挂载，导致解析错误

**修复**: 
```html
<!-- 修改前 -->
<body :data-theme="isDark ? 'dark' : 'light'">

<!-- 修改后 -->
<body data-theme="dark">
```

同时在Vue组件中动态更新body属性：
```javascript
// mounted钩子中设置初始主题
mounted() {
    document.body.setAttribute('data-theme', this.isDark ? 'dark' : 'light');
    this.startAutoRefresh();
}

// toggleTheme方法中更新主题
toggleTheme() {
    this.isDark = !this.isDark;
    storage.setTheme(this.isDark ? 'dark' : 'light');
    document.body.setAttribute('data-theme', this.isDark ? 'dark' : 'light');
    if (this.chart) {
        this.chart.applyOptions(this.getChartOptions());
    }
}
```

**验证**: 
- ✅ 主应用加载正常
- ✅ 页面正常显示
- ✅ 主题切换正常工作
