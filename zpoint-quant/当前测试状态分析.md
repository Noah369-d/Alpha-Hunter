## 🔍 SignalGenerator 问题详细分析

### 需要检查的文件

1. **src/utils/SignalGenerator.js**
   - 检查 export 语句
   - 确认是 `export default class SignalGenerator`

2. **src/utils/SignalGenerator.test.js**
   - 检查 import 语句
   - 确认是 `import SignalGenerator from './SignalGenerator.js'`

3. **src/utils/SignalGenerator.property.test.js**
   - 同样检查 import 语句

### 可能的问题

1. **导出问题**:
   ```javascript
   // 错误
   export { SignalGenerator as default }
   
   // 正确
   export default SignalGenerator
   ```

2. **导入问题**:
   ```javascript
   // 错误
   import { SignalGenerator } from './SignalGenerator.js'
   
   // 正确
   import SignalGenerator from './SignalGenerator.js'
   ```

3. **模块加载问题**:
   - 可能是循环依赖
   - 可能是 IndexedDB 初始化问题
